Сервис PSU Report Builder является программным средством, позволяющим автоматизировать процесс формирования отчётной информации по подразделениям учебного заведения. Данный сервис автоматизирует процесс анализа, получения данных и формирования готовых отчётов о НИД подразделений ПГУ. В ходе работы сервис выполняет следующие задачи:

  1. Поиск информации;
  2. Анализ и фильтрация полученных данных;
  3. Структурирование данных по заданным формам;
  4. Формирование отчётной документации по НИР сотрудников и аспирантов;
  5. Обеспечение интерактивной настройки параметров выходных данных;
  6. Обеспечение настройки формы отчётного документа.
Предполагается возможность подключения ресурсов Scopus и eLibrary для расширения поисковых запросов.
